# Website Maintenance Guide

This file documents which parts of the site go stale and how to update them.

---

## Site structure

| File | Contents |
|---|---|
| `index.md` | Homepage: intro, research highlights, news |
| `research.md` | 5 research topic sections with paper lists |
| `people.md` | PI bio, current members, alumni table |
| `publications.md` | Full paper list organized by topic |
| `software.md` | Green, ALPS, ALPSCore descriptions and logos |
| `positions.md` | Postdoc, PhD, master/bachelor openings |
| `_config.yml` | Theme, title, navigation |

---

### 4. Open positions (`positions.md`)

Positions are funded by the ERC Advanced grant *Quantum Algorithms*. Update if the grant ends or positions are no longer available.

**PhD timeline** — the numbered timeline is specific to the 2026–2027 cycle and must be updated each year. Check https://szkolydoktorskie.uw.edu.pl/en/ for the new cycle's dates (typically published in early spring) and update both the dates and the recruitment page link.
